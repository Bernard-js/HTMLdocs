Representing computer code

elements for marking up computer code:

  • <code>: For marking up generic pieces of computer code.
  • <pre>: For retaining Whitespace (usually blocks of code).
  • <var>: For marking up variable names.
  • <kbd>: For marking up keyboard inputs, like shortcuts.
  • <samp>: For marking the output of a computer program.

see that example below: 

            
                var number = 25
                console.log(number)

use the console.log() method to output information in the console.

in the example above, number represents the number 25.

copy the code above with Ctrl+C.
